# StationSpotter
Tells you when and where the ISS will appear in your local sky!

## Description

StationSpotter is an application that will tell you when and where to look at your local sky to see the ISS pass by!

## Getting Started

### Dependencies

* Currently, this project requires Python 3.12. Eventually, it will run inside of a docker container and not be platform dependent.
* This project also requires you to register for an API token from N2Y0. Once registered, you can place your Token inside of config.ini

### Installing

* To install simply clone this repository!
```
git clone https://github.com/cackler12/StationSpotter
```

### Executing program
```
/bin/python3 /home/user/StationSpotter/main.py
```
